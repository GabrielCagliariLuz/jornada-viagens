# Jornada Viagens ✈️

[🔗 Acesse clicando aqui](https://gabrielcagliariluz.github.io/jornada-viagens/)

O Jornada Viagens é uma landing page responsiva desenvolvida para uma agência de viagens fictícia. 
O foco deste projeto foi construir uma interface atraente e amigável, garantindo uma excelente experiência de navegação tanto em dispositivos móveis quanto em desktops. 
A estrutura do código foi modularizada em diferentes arquivos CSS para facilitar a manutenção, evitar conflitos de especificidade e manter a organização.
